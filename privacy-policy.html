Privacy Policy

This Privacy Policy explains how HariQu collects, uses, stores, and protects user information. By using HariQu, users agree to the practices described in this policy.

1. Application Information

HariQu is a personal productivity and communication application that may include features such as profile management, daily checklist, study tracker, diary notes, expense tracking, activity map, friend connection, and realtime chat.

2. Information We Collect

HariQu may collect and store the following types of information:

  • Account information, such as user ID, display name, email address, phone number, and profile photo URL.
  • User-generated content, such as diary notes, checklist items, study tracker entries, expense records, chat messages, and friend requests.
  • App preference data, such as selected language, theme mode, and currency preference.
  • Location-related information if users choose to use activity map or location-based features.
  • Technical information needed to operate authentication, database, and app functionality.

3. How We Use Information

The information is used to provide and improve HariQu features, including user authentication, personal data management, realtime chat, friend request handling, activity tracking, and app personalization.

4. Data Storage and Security

HariQu uses Firebase services, including Firebase Authentication and Cloud Firestore, to support account access and data storage. Access to user data is controlled using authentication and database security rules.

5. Data Sharing

HariQu does not sell user personal data. User data is not shared with third parties for advertising purposes. Data may be processed by service providers used to operate the application, such as Firebase infrastructure.

6. Chat and Friend Features

HariQu may allow users to connect with other users and exchange chat messages. Chat-related data may include sender ID, receiver ID, message text, read status, reaction data, and timestamps.

7. User Control and Data Deletion

Users may request account and data deletion. Instructions are available on the Account Deletion Information page.

8. Children’s Privacy

HariQu is not intended for children under the minimum age required by applicable laws. If we become aware that personal data from a child has been collected without appropriate consent, we will take reasonable steps to delete it.

9. Changes to This Policy

We may update this Privacy Policy from time to time. Updates will be posted on this page with a revised date.

10. Contact

For privacy-related questions or requests, please contact the HariQu developer through the official support contact listed in the Google Play Store listing.
